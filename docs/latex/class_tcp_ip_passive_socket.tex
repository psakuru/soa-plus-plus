\subsection{Detailed Description}
Socket TCP/IP che accetta richieste di connessione. 

La classe \hyperlink{class_tcp_ip_passive_socket}{TcpIpPassiveSocket} mantiene un descrittore di file relativo al socket passivo gestito. Un socket passivo è un socket su cui si effettua l' operazione di listen.
