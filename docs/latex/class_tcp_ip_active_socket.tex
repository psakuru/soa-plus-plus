\subsection{Detailed Description}
Socket TCP/IP per comunicazione attiva. 

La classe \hyperlink{class_tcp_ip_active_socket}{TcpIpActiveSocket} mantiene un descrittore di file relativo al socket attivo gestito. Un socket attivo Ã¨ un socket su cui si effettua una comunicazione attiva, a differenza di quanto accade quando si effettua l' operazione di listen. Socket attivi sono socket su cui si fa la connect e socket ritornati dalla accept.
